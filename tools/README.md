# 字幕搜索音频截取工具 v2.0

## 功能

| 功能 | 说明 |
|------|------|
| 🔍 **关键词搜索** | 支持正则表达式搜索所有 SRT 字幕 |
| 🎯 **10个快速场景** | 打招呼、问路、餐厅、医院、家庭、谈判、化学、情感、疑问、保险 |
| ☑️ **多选** | 点击列表切换选中，支持全选/反选 |
| ▶️ **音频预览** | 点击 `>` 或双击片段，在默认播放器中试听 |
| ✂️ **三种截取模式** | 字幕行 / 扩展N秒 / 固定时长 |
| 🔗 **合并导出** | 多个片段合并为一个 MP3 文件 |
| 📦 **批量提取** | 一次性提取多个片段 |
| ⚙️ **音频设置** | 码率 32k-128k、单声道优化 |
| 💾 **配置记忆** | 自动保存路径和设置 |

## 使用方法

```bash
cd D:\EnglishPracticeMiniProgram\tools
python subtitle_audio_extractor.py
```

### 预览功能

- 点击列表中的 `>` 按钮 → 生成5秒预览并播放
- 双击列表行 → 同样可以预览
- 预览使用低码率(32k)临时文件，不占用输出目录

### 合并导出

1. 选择多个片段（至少2个）
2. 在右侧输入合并文件名
3. 点击 **"合并导出"** 按钮
4. 程序会：
   - 自动提取所有片段
   - 使用 FFmpeg concat 协议合并
   - 生成一个完整的 MP3 文件
   - 完成后自动打开输出目录

### 文件名模板变量

| 变量 | 说明 | 示例 |
|------|------|------|
| `{index}` | 编号（3位） | 001 |
| `{episode}` | 集数 | S01E01 |
| `{timestamp}` | 时间戳 | 021500 |
| `{text}` | 文本摘要 | what_are_you |
