## 项目概述

基于尚雯婕英语学习法开发的英语听写练习小程序，采用游戏化关卡设计，支持音频播放、AB复读、手写文本扫描识别、智能对照批改等功能。

## 使用说明

### 1. 在微信开发者工具中打开
1. 打开微信开发者工具
2. 点击「导入项目」
3. 选择 `D:\EnglishPracticeMiniProgram` 目录
4. 点击「编译」

### 2. 功能流程
```
关卡列表 → 选择关卡 → 音频播放 → 扫描文本 → 对照批改 → 获得积分
```

### 3. 核心操作
- **播放控制**: 点击播放按钮，可拖动进度条
- **AB复读**: 开启复读开关，分别点击"设置A点"和"设置B点"
- **速度调节**: 点击 0.5x/0.75x/1.0x/1.25x/1.5x 按钮
- **扫描识别**: 点击"扫描文本"，拍照或从相册选择
- **手动输入**: 如果识别效果不佳，可选择手动输入
