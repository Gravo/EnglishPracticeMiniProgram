## 项目简介

### 核心功能
基于尚雯婕英语学习法，结合腾讯云OCR和AI技术，打造智能化英语听写练习小程序。

**主要功能：**
1. **关卡式学习** - 5个难度递增的关卡，涵盖政治历史题材
2. **AI音频播放** - 支持AB复读、变速播放、进度控制
3. **智能OCR识别** - 手写听写文本扫描识别（腾讯云OCR）
4. **AI对照批改** - 基于编辑距离的文本对比算法
5. **游戏化积分** - 积分、等级、成就系统

### 技术架构
- **前端**: 微信小程序原生框架
- **AI/OCR**: 腾讯云OCR文字识别
- **存储**: 微信云开发（CloudBase）
- **算法**: Levenshtein距离文本对比

## AI能力应用

### 1. OCR文字识别（已接入腾讯云）
- **应用场景**: 用户手写听写文本扫描识别
- **技术方案**: 腾讯云通用印刷体识别API
- **准确率**: >85%英文手写识别

### 2. 混元大模型可接入点

#### A. 智能批改增强
- **需求**: 不仅指出错误，还能解释错误原因
- **混元应用**: 
  - 输入：用户文本 vs 原文
  - 输出：错误分析 + 语法讲解 + 改进建议

#### B. AI生成听力材料
- **需求**: 根据用户水平自动生成听力文本
- **混元应用**:
  - 输入：难度等级、主题偏好
  - 输出：定制化听力文本 + 音频脚本

#### C. 智能对话练习
- **需求**: 听写后的口语对话练习
- **混元应用**:
  - 基于听力材料生成对话场景
  - AI角色扮演对话练习

#### D. 个性化学习建议
- **需求**: 根据用户错误模式给出学习建议
- **混元应用**:
  - 分析用户历史错误
  - 生成个性化学习计划
