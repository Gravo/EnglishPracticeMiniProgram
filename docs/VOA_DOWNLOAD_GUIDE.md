## 手动下载步骤

### 1. 访问网站
打开 https://www.manythings.org/voa/

### 2. 搜索主题
- 使用Ctrl+F搜索关键词：
  - "Middle East"
  - "Israel"
  - "Iran"
  - "Iraq"
  - "oil"
  - "war"

### 3. 下载音频
- 点击文章标题
- 找到MP3链接
- 右键 → "另存为"
- 保存到 D:\downloads_english\

### 4. 整理文件
```
D:\downloads_english\
├── middle_east_history\
│   ├── 01_oil_history.mp3
│   ├── 02_israel_creation.mp3
│   └── ...
├── modern_conflicts\
│   ├── 01_gulf_war.mp3
│   └── ...
└── transcripts\
    ├── 01_oil_history.txt
    └── ...
```
