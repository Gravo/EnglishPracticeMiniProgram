# 英语听写练习小程序 - 产品需求规格书 (PRD)

**版本**: v1.0  
**日期**: 2026-03-21  
**状态**: 需求分析阶段

---

## 1. 产品概述

### 1.1 产品背景
基于尚雯婕英语学习法（听写训练法），开发一款游戏化英语听写练习小程序。用户通过反复听写音频材料，提升英语听力水平。

### 1.2 核心功能
- 关卡式音频听写练习
- 音频播放/复读/暂停控制
- 手写文本扫描识别
- 听写结果对照批改

### 1.3 目标用户
- 英语学习者
- 备考四六级/雅思/托福的学生
- 需要提升听力的职场人士

## 3. 非功能需求

### 3.1 性能需求
- 音频加载时间 < 2秒
- 页面切换时间 < 500ms
- OCR识别时间 < 3秒

### 3.2 兼容性需求
- 支持 iOS 12+ / Android 8+
- 微信小程序基础库 2.30.0+

### 3.3 数据存储
- 本地存储：播放进度、关卡状态
- 云端存储：成绩记录、排行榜

## 4. 界面原型

### 4.1 页面结构
```
首页 (关卡列表)
  └── 关卡详情页
        ├── 音频播放页
        ├── 扫描识别页
        └── 对照批改页
```

### 4.2 关键界面描述

**关卡列表页**
- 顶部：用户等级、积分、连续打卡天数
- 中部：5个关卡卡片，横向或纵向排列
- 底部：设置、排行榜入口

**音频播放页**
- 顶部：关卡标题、返回按钮
- 中部：音频波形/进度可视化
- 底部：播放控制区(播放/暂停/复读/速度)
- 底部：扫描按钮(跳转到扫描页)

**扫描识别页**
- 相机预览区域
- 拍摄按钮
- 框选工具(拍摄后)
- 确认/重拍按钮

**对照批改页**
- 左右/上下分栏显示原文vs用户文本
- 错误高亮(红色=错误，黄色=遗漏)
- 评分展示
- 重新听写/下一关按钮

## 6. 技术方案

### 6.1 音频处理
- 微信小程序 `InnerAudioContext` API
- 支持播放进度控制、循环播放

### 6.2 OCR识别
- 方案A: 微信OCR插件
- 方案B: 腾讯云OCR API
- 方案C: 百度OCR API

### 6.3 文本对比算法
- 基于编辑距离(Levenshtein Distance)
- 支持单词级别对比
