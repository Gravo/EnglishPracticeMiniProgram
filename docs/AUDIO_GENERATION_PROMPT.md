# 英语听写练习小程序 - 音频素材生成 Prompt

## 使用说明

将以下 Prompt 复制发送给 **豆包** 或 **DeepSeek**，生成音频和文本素材。

---

## Level 1: 中国古代四大发明 (简单/60秒)

```
请为我生成一段适合英语初学者的听力材料。

【主题】中国古代四大发明 (The Four Great Inventions of Ancient China)

【要求】
1. 语速：慢速，清晰发音，适合初中英语水平
2. 时长：约60秒
3. 词汇量：基础词汇，避免生僻词
4. 内容要点：
   - 介绍中国有四大发明
   - 分别是造纸术、印刷术、火药、指南针
   - 造纸术发明于汉代
   - 这些发明对世界的影响

【输出格式】
1. 英文原文（约80-100个单词）
2. 中文翻译
3. 关键词汇表（单词+音标+中文释义）
4. 句子分割（每句话单独一行）

【额外要求】
如果可能，请用TTS工具生成MP3音频文件，或提供适合朗读的文本标记（如停顿、重音）。
```

## 推荐 TTS 工具

如果豆包/DeepSeek 无法直接生成音频，可以使用以下 TTS 工具：

1. **Azure TTS** (推荐) - 语音自然，支持多种风格
2. **Google Cloud TTS** - 质量高，支持SSML
3. **Amazon Polly** - 性价比高
4. **讯飞配音** - 中文支持好
5. **剪映** - 免费，适合简单需求

---

## 音频质量要求

- 格式：MP3
- 采样率：44.1kHz 或 48kHz
- 比特率：128kbps 或更高
- 声道：单声道或立体声
- 音量：标准化至 -3dB
